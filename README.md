# 🎰 幸运大转盘

一个精美的抽奖转盘网页应用，支持自定义奖品、抽奖历史记录和彩带庆祝效果。

![Preview](https://via.placeholder.com/400x400?text=Lucky+Wheel)

## ✨ 功能特点

- 🎨 **精美界面** - 8个彩色扇区，流畅旋转动画
- 🎯 **公平抽奖** - 随机算法确保公平
- ⚙️ **自定义奖品** - 支持添加、删除、修改奖品
- 📜 **历史记录** - 自动记录抽奖历史（最多50条）
- 🎉 **彩带效果** - 中奖时彩带庆祝动画
- 🔊 **音效反馈** - 旋转时播放音效
- 📱 **响应式设计** - 完美适配桌面和移动设备
- 💾 **本地存储** - 使用 LocalStorage 保存数据

## 🚀 快速开始

### 本地运行

直接双击 `index.html` 即可在浏览器中运行。

### GitHub Pages 部署

1. Fork 或 Clone 本仓库
2. 在 GitHub 仓库设置中启用 Pages
3. 选择 `main` 分支作为源
4. 访问 `https://你的用户名.github.io/spin-wheel-game/`

## 🎮 使用方法

1. 点击中心的"开始抽奖"按钮
2. 转盘旋转4秒后停止
3. 弹窗显示中奖结果
4. 可在设置面板自定义奖品

## ⚙️ 自定义奖品

- 点击"添加奖品"增加新奖品
- 点击"删除"移除奖品（至少保留2个）
- 点击"重置默认"恢复初始配置
- 修改会自动保存到本地
